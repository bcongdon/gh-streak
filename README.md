# gh-streak

A Python library and API for fetching GitHub streaks.

## Python
### Usage
```python
import ghstreak

# Get the current streak for csu
ghstreak.get_streak_for_user('csu')
```

### Installation
```bash
pip install ghstreak
```

## API
### Installation
```bash
cd api
pip install -r requirements.txt
```

### Usage
```bash
cd api
python server.py
```
