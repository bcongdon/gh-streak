# gh-streak

A Python library and API for fetching GitHub streaks.

## Python
### Usage
```python
import ghstreak

# Get the current streak for csu
ghstreak.get_streak_for_user('csu')
```

### Installation
```bash
pip install ghstreak
```

## API
### Usage
```bash
# In the api directory
python server.py
```

### Installation
```bash
# In the api directory
pip install -r requirements.txt
```
